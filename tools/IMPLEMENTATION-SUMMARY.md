## ✨ 技术特点

### 零依赖架构
- ✅ 无需 npm/yarn 等包管理器
- ✅ 无需 webpack/babel 等构建工具
- ✅ 无需任何 JavaScript 框架 (React/Vue/Angular)
- ✅ 无需任何 CSS 框架 (Bootstrap/Tailwind)
- ✅ 可选 CDN 库仅用于高级功能 (PDF/GIF 处理)

### Web API 充分利用
```javascript
核心 API 使用统计:
├─ Canvas API (8 个工具)
│  ├─ 图片压缩、格式转换、加水印
│  ├─ 黑白化、像素化、二维码生成
│  ├─ LED 动画、绘制等
│
├─ Web Speech API (1 个工具)
│  └─ 文本转语音
│
├─ Clipboard API (多个工具)
│  └─ 复制粘贴功能
│
├─ URL API (3 个工具)
│  ├─ URL 编码、URL 解析、URL 操作
│
├─ Date/Number API (4 个工具)
│  ├─ 时间戳转换、世界时间、日期计算
│
├─ MediaRecorder API (1 个工具)
│  └─ 在线录屏
│
└─ 其他 API (12 个工具)
```

## 💡 核心实现亮点

### 1. 数据驱动架构
```javascript
const CATEGORIES = [...]    // 12 个分类
const TOOLS = [...]         // 170+ 个工具定义

// 自动生成导航、卡片、功能
renderSidebar()   // 从 CATEGORIES
renderTools()     // 从 TOOLS
```

### 2. 实时搜索与过滤
```javascript
搜索输入 → 输入事件 → 过滤 TOOLS → 重新渲染
分类选择 → 点击事件 → 过滤分类 → 重新渲染
组合过滤: 分类 + 搜索
```

### 3. 响应式三层设计
```
桌面 (≥1024px)
├─ 左侧边栏 (固定)
├─ 上方搜索
└─ 工具网格 (4+ 列)

平板 (768px)
├─ 抽屉式菜单
├─ 搜索框
└─ 工具网格 (3 列)

手机 (<480px)
├─ 隐藏菜单 + 按钮
├─ 全屏搜索
└─ 工具网格 (2 列)
```

### 4. 统一的工具页面模板
```html
<!DOCTYPE html>
<head>
  <link rel="stylesheet" href="../../styles/main.css">
</head>
<body>
  <div class="tool-page">
    <div class="tool-topbar">
      <a href="../../index.html" class="back-btn">← 返回</a>
    </div>
    <div style="flex:1; padding: 24px;">
      <!-- 工具内容 -->
    </div>
  </div>
</body>
```

所有 37 个工具都遵循此模板，保证一致的用户体验。

- 所有工具使用原生 Web API

- 任何浏览器都可直接打开运行
